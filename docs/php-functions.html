PHP Functions

This is an overview of useful PHP functions and variables provided by LaceKit. They let you work with data easily and make your prototype feel real.

Function Reference

Function Description and Parameters
__() (Two underscores) Returns the translation for the provided key. Returns the value of the key when no translation is found.

Parameters:
- key: The key to translate
- language (optional): The language to translate to. Defaults to current active language

Usage: __($key) or __($key, $language)
alert() Builds the HTML for a Bootstrap Alert.

Parameters:
- text: The alert text (can contain HTML)
- class (optional): Bootstrap alert modifier class (info/success/warning/danger)
- icon (optional): Font Awesome icon name
- id (optional): Custom alert ID
- dismiss (optional): Boolean to make alert dismissable (default: true)

Usage: alert($text) or alert($text, $class)
forceLogin() Forces user login to access the requested page. Shows login form if user is not logged in.

Usage: forceLogin()
getDeeplink() Returns link to current page including GET Parameters and automatic user login. Useful for role-based prototypes.

Usage: getDeeplink()
getFirstUserLogin() Gets the email of the first user specified in users.yml file.

Usage: getFirstUserLogin()
getUniqueId() Returns an incremented unique identifier number.

Parameters:
- variableName (optional): Variable name for unique ID tracking

Usage: getUniqueId() or getUniqueId($variableName)
hideIf() Outputs the class “hide” if current user is in specified usergroups.

Parameters:
- usergroups: Comma-separated list of affected usergroups

Usage: hideIf("admin, superadmin")
includeIf() Includes specified file if current user is in specified usergroups.

Parameters:
- usergroups: Comma-separated list of affected usergroups
- file: Path to file to include

Usage: includeIf("admin", "path/to/file")
makeDateFromString() Converts text description to date format.

Parameters:
- string: Text datetime description
- format (optional): PHP date format pattern

Usage: makeDateFromString("last Thursday")
reorder() Reorders array by specified key values.

Parameters:
- array: Array to reorder
- column: Key to sort by
- direction (optional): “asc” or “desc” (default: asc)

Usage: reorder($array, $column)
showIf() Opposite of hideIf() - shows content for specified usergroups.

Parameters:
- usergroups: Comma-separated list of affected usergroups

Usage: showIf("admin, superadmin")
today() Returns today’s date in configured format.

Parameters:
- format (optional): PHP date format pattern

Usage: today() or today("Y-m-d")
tomorrow() Returns tomorrow’s date in configured format.

Parameters:
- format (optional): PHP date format pattern

Usage: tomorrow() or tomorrow("Y-m-d")
yesterday() Returns yesterday’s date in configured format.

Parameters:
- format (optional): PHP date format pattern

Usage: yesterday() or yesterday("Y-m-d")

Variables

Variable Description
$lastUniqueId Returns the last value assigned by getUniqueId()
$currentYear Returns current year as four digit number
$loggedIn Boolean indicating if user is logged in
$justloggedIn Boolean true only on page called by login form
$activeUser Array of current active user data (false if not logged in)
$username Active user’s username
$usermail Active user’s email
$userrole Active user’s role
$userpermissions Array of user permissions from users.yml
